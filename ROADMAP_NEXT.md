## 🥇 1. 서술형 문제 생성 & AI 채점 (최우선)
**배경**: 내신이 서술형 비중 급증. 랜딩 후킹멘트("외우지 말고 생각하게")의 핵심 근거.

**범위**:
- AI가 본문·대화문 기반 **서술형 문항 생성** (영작 / 요약 / 이유 설명 / 빈칸 서술)
- **AI 채점**: 모범답안 대비 의미 일치도 + 문법 피드백 + 부분점수
- 핵심 키워드 채점 루브릭
- 오답노트·약점처방전 연동
- 문제뱅크에 "서술형" 유형으로 적재
- 시험 범위 선택(단어/대화문/본문/전체)에 자연스럽게 포함

## 🥈 2. 학생 회원가입 절차 개편
**배경**: 현재 학생 가입에 이메일·연락처 입력란 없음. 선생님 코드 없으면 가입 불가.

**변경 사항**:
- 학생 가입 폼에 **이메일 + 연락처(전화번호)** 입력란 추가
- **선생님 코드 없어도 가입 가능** (개인 회원)
- **하나의 학생 가입 페이지**에서 분기:
  - 선생님 코드 입력 O → 해당 **선생님 플랜 등급**으로 자동 연결 (선생님이 PRO면 학생도 혜택)
  - 선생님 코드 입력 X → **FREE 개인 회원** → 결제(STUDENT 4,900원) 시 유료 기능

**구현 포인트**:
- `auth.sign_up()` 에 email/phone 필드 + 선택적 join_code
- 코드 있으면 teacher_id 연결 + plan 상속 / 없으면 plan='free'
- profiles에 phone 컬럼 추가 (마이그레이션)
