```
[에픽1 배포 안정화] ──전제──▶ [에픽2 시험 요약노트]  (오늘 메인)
```

## 🟦 에픽 2 · 시험 요약노트 재설계 (오늘 메인) — 건의 대부분 수렴
> 현재: AI가 앞/뒷면 2p에 자동배치 → 통제 불가 = "이상하게". 아래 4레이어 **순차**.

### 2-1. 데이터 레이어 — 무엇을 위로 올릴까
- **오답·출제 빈도순 정렬** (교육팀 J.H.) — 그 학생이 자주 틀린 것/빈출을 상단에
- 소스: 기존 오답노트·문제뱅크 데이터 재활용

### 2-2. 생성 레이어 — 섹션 선택
- **체크박스로 섹션 선택**: 단어/문법/대화·문장/본문/패턴/비법/시험포인트
- **선택한 것만 AI 생성** → UX + AI 비용 절감 (D.Y.)
- front/back 자동분배 구조 폐기

### 2-3. 렌더 레이어 — 출력물
- **여백 5mm** (현재 8mm)  ※ K.D. 경고와 충돌 가능 → 2-4에서 검증
- **한 단원 = A4 한 바닥** (2p 고정 폐기, 넘치면 자동 흐름)
- **정보 밀도 ↑** (다단, 간격 축소)
- **흑백 출력 대응** (색 배경 헤더 → 흑백서 뭉개짐, K.D.)
- **빈칸(셀프테스트) 모드 — 기본 OFF** (심리팀 H.S.: 시험 직전 멘붕 방지)
- **하단 작은 '반반 BanBan' 브랜딩** (그로스 N.W.)

### 2-4. 검증 레이어 — 🚨 필수
- **실제 PDF로 뽑아 인쇄 영역 확인** (K.D.) — 5mm가 프린터에서 잘리는지
- 잘리면 5mm ↔ 인쇄가능여백 절충안 회원님과 결정
